# ❤️ Homenagem à Geovana Nascimento

Este repositório contém o código de um site romântico criado como uma singela homenagem à minha amada esposa, **Geovana Nascimento**. 

O projeto é uma celebração do nosso amor, reunindo mais de 100 fotos que marcam a nossa jornada juntos e expressam o quanto sou grato por tê-la ao meu lado como minha companheira de vida.

## ✨ Sobre o Projeto
O site foi desenvolvido com tecnologias web modernas para proporcionar uma experiência visual "Premium" e emocionante, incluindo:
- **Galeria Dinâmica:** Exibição de 106 memórias inesquecíveis.
- **Design Romântico:** Estética sofisticada com animações de corações e efeitos suaves.
- **Fundo Musical:** Integrado com a API do YouTube para tocar trilhas sonoras que marcaram nossa história.
